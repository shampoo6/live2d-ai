
============================================================

Sample Model
Miara (PRO version)

============================================================
  It is a model to learn how to use drawing order groups and animate whole body.

  Besides the main character Miara,
  Fairy(drawing order groups used) and moving background are applied into a single model.
  Wraparound front and back of the character, walking etc.,
  You can also learn dynamic animation using the whole model.
